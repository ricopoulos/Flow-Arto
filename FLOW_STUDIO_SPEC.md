# Flow Studio – Natural-Language Design Composer (Web-First Spec)

## Purpose
Turn plain English intent into an elegant landing page mock (HTML/CSS), using small agents and web-safe tools inside **Claude Code Web**. Start minimal; no external installs.

## Outcomes
- Design brief from intent (audience, tone, goals)
- Layout map + copy skeleton
- Style tokens (palette, typography, spacing, radii, shadows)
- Accessible, responsive **single-page** HTML/CSS mock
- Short rationale and next steps (A/B ideas)

## Constraints (Web)
- **No CLI assumptions**, no package installs unless I approve.
- Prefer **vanilla HTML/CSS** or **Tailwind CDN**; keep CSS small.
- Use **Playwright MCP** only if already available; screenshots: JPEG, quality≈60, max 1200×800, ≤3 total; save paths (don’t inline).
- Idempotent edits; show diffs; ask before large refactors.
